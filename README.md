# goals-front

UI управления целями.

## dev-разработка

[Frontend login](http://localhost:5173/login) <br />

### Быстрый старт

Перед началом работы: `nvm use`, так как проект использует Node.js `24.0.0`.

```bash
npm install
npm run dev
```

### Переменные окружения

Для экрана входа используются переменные:

- `VITE_APP_TITLE` - название приложения на экране входа.
- `VITE_APP_ID` - идентификатор приложения для auth-сервиса.
- `VITE_GOALS_AUTH_API` - базовый URL сервиса авторизации.

### Документация

- Техническая документация фронтенда: текущий `README.md`.
- Проектная документация (ТЗ): Docusaurus-сайт в папке `docs/`.

Запуск Docusaurus локально:

```bash
npm run docs:start
```

### Текущий scope

- `goals-front` разрабатывается как самостоятельное приложение на MUI.
- Общие компоненты пока остаются внутри этого репозитория и стилизуются через единую тему.
- Разводящая страница `Melkor Apps` пока не реализуется.
- После появления второго приложения и подтвержденного дублирования в отдельный пакет будут вынесены только нейтральные UI-компоненты, с публикацией в Verdaccio.
