# Техническое задание (ТЗ) для проекта "goals-front"

Разделы ТЗ, описанные на текущем этапе:

1. [Авторизация и экран входа](/auth-login)

## Текущие границы реализации

- Сначала разрабатывается `goals-front` как самостоятельное приложение на MUI.
- Общие компоненты остаются внутри этого репозитория и используют единую тему.
- Разводящая страница `Melkor Apps` пока не входит в этап реализации.
- После появления второго приложения и подтвержденного дублирования в отдельный пакет будут вынесены только нейтральные UI-компоненты с публикацией в Verdaccio.
